## Open－Mail－System

### 主要特性
+ 由服务端代理发件，避开使用mail()等函数
+ 发件全过程可追溯，便于管理员监控
+ 便于管理，只用一个Email以及开放API即可完成所有的邮件发送任务

### 使用
+ 在**smtp.php**文件中按照文件注释修改配置
+ 导入**sql.sql**到数据库中
+ 修改**sql.php**文件中的数据库设置
